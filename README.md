# 🛒 쇼핑 리스트 앱 (Shopping List App)

단일 HTML 파일로 동작하는 간단한 쇼핑 리스트 웹 앱입니다. 별도의 설치나 빌드 없이 브라우저에서 바로 열어 사용할 수 있습니다.

## 기능

- 아이템 추가 / 삭제
- 체크박스로 구매 완료 표시 (취소선)
- 남은 항목 개수 카운터
- 완료된 항목 일괄 삭제
- `localStorage` 기반 자동 저장 (새로고침해도 데이터 유지)

## 사용 방법

`index.html` 파일을 브라우저로 엽니다.

## 기술 스택

순수 HTML / CSS / JavaScript (외부 의존성 없음)
